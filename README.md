# Helix Monitor

A modern Linux system monitor built with Electron, React, and TypeScript.

## Features

* Real-time CPU usage monitoring
* Memory usage statistics
* Disk usage information
* System uptime and OS details
* Modern, lightweight interface
* Linux AppImage distribution

## Installation

Download the latest AppImage from the Releases page.

Make it executable:

```bash
chmod +x *.AppImage
```

Run:

```bash
./*.AppImage
```

## Development

Clone the repository:

```bash
git clone https://github.com/BrainNOTrot/helix-monitor.git
cd helix-monitor
```

Install dependencies:

```bash
npm install
```

Run in development mode:

```bash
npm run dev
```

Build:

```bash
npm run build
```

Create distributables:

```bash
npm run package
```
